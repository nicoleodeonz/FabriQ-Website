🚀 GET STARTED IN 3 STEPS:

Step 1: Ensure MongoDB is Running
  → Windows: MongoDB should start automatically as a service
  → Check Task Manager: Services → "MongoDB Server" (should be Running)
  → Mac/Linux: Run `mongosh` in terminal

Step 2: Start Both Servers (Choose One):
  
  OPTION A - Windows (Easiest):
  ┌─────────────────────────────────────────────────────┐

🐛 IF YOU RUN INTO ISSUES:

  MongoDB not running?
  → Windows: Start service in Services.msc
  → Mac: brew services start mongodb-community
  
  Port already in use?
  → Backend (5000): Close other apps using that port
  → Frontend (5173): Change port in vite.config.ts
  
  Missing modules?
  → Run: npm install (in the directory)
  
  Need help?
  → See [QUICK_START.md](QUICK_START.md) troubleshooting
